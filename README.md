# Math_Timed_Quiz
A modern web-based math training application that generates random math problems, tracks response time, and measures user accuracy. The application features a modern dark glossy UI with red accents for an engaging learning experience. This project is designed to help users improve their mental math speed and accuracy through interactive quizzes.

## Features

🎯 Multiple difficulty levels (Easy, Medium, Hard)

🔢 Random math problem generation

⏱ Response time tracking

📊 Accuracy calculation

🎨 Modern dark glossy UI

## 🖥️ Technologies Used
- HTML5 (Python Logic)

No frameworks/pipeline required
